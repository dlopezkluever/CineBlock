  My Recommendations

  Start with these ~12 for max coverage with min effort:

  1. Box (already have)
  2. Cylinder
  3. Sphere
  4. Plane/Flat
  5. Capsule/Rod
  6. Table
  7. Chair
  8. Car
  9. Tree
  10. Wall segment
  11. Stairs
  12. Dog

  The geometric primitives (1-5) are trivial — they're just Three.js built-in geometries. The silhouette shapes      
  (6-12) would need simple procedural geometry or very low-poly constructions from combined primitives.

  Key UX question: Should these be categorized in the picker (Primitives / Furniture / Vehicles / Nature / etc.) or  
  just a flat icon grid? For 12 shapes a flat grid works; if we go 20+ we'd want categories.
